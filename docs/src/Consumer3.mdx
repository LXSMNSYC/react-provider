# Consumer3

`<Consumer3>` component is similar to `<Consumer>` component but it can consume values from three nearest `<Provider`> ancestors.

## Basic Usage

<Playground>
  <MultiProvider
    providers={[
      <Provider of="A" value="Foo" />,
      <Provider of="B" value="Bar" />,
      <Provider of="C" value="Baz" />,
    ]}
  >
    <Consumer3 of={['A', 'B', 'C']} builder={(x, y, z) => <h1>{x} {y} {z}</h1>} />
  </MultiProvider>
</Playground>
