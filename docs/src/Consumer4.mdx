# Consumer4

`<Consumer4>` component is similar to `<Consumer>` component but it can consume values from four nearest `<Provider`> ancestors.

## Basic Usage

<Playground>
  <MultiProvider
    providers={[
      <Provider of="A" value="Fee" />,
      <Provider of="B" value="Fi" />,
      <Provider of="C" value="Fo" />,
      <Provider of="D" value="Fum" />,
    ]}
  >
    <Consumer4 of={['A', 'B', 'C', 'D']} builder={(x, y, z, w) => <h1>{x} {y} {z} {w}</h1>} />
  </MultiProvider>
</Playground>
